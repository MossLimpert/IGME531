Brief, One-Page Summary:

Paper.js is a library built around making vector shapes and infinitely scalable graphics. It is almost stupefyingly simple, befuddling my brain, which was trying to invent complexity where there was none. The library is well documented, and when you use npm to put it into your project, it includes a ton of helpful examples that helped me accomplish my goal (draw shapes, then expor a svg). Even though I prefer to use cdns to downloading a whole library, the extra examples and support make it worth downloading. The shortcoming I found is that you have to use var and not let? I am confused by this, but I guess it is because it is compiled in its own scope.

Method Instructions:

  • Create basic html project
  • Use node package manager to install Paper.js into the directory of the project. npm install paper
  • Create the js file that will house my Paper drawings, and link it to my html page as script type text/paperscript
  • Fill my script with shapes and paths!
